# tic-tac-toe-python
A simple Tic Tac Toe game built in Python using basic game logic and user input.

This is a simple Tic Tac Toe game built using Python as a beginner milestone project.  

○ Features

• Turn-based gameplay  
• User input handling  
• Detects if a space is already taken  
• Shows message for invalid row number or index  
• Win and draw detection  
• Basic game logic using conditions and loops

○ What I learned

• Python fundamentals  
• Functions and loops  
• Conditional statements  
• Problem-solving and game logic
